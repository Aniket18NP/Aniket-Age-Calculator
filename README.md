# Age Calculator

A simple and responsive Age Calculator built with HTML, CSS, and JavaScript that accurately calculates a user's age based on their birth date.

## 📌 Features

* Calculate exact age in years, months, and days
* User-friendly interface
* Responsive design for all devices
* Fast and accurate calculations
* Input validation for dates

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

1. Clone the repository:

```bash
git clone https://github.com/Aniket18NP/aniket-age-calculator.git
```

3. Run `index.html` in your browser.

## 🎯 How to Use

1. Enter your date of birth.
2. Click the **Calculate Age** button.
3. View your exact age instantly.
